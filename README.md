# Lunaue
Luau Syntax Analysis Toolkit Library in C
## Currently Available
- Lexer/Token Generator
## Unavailable/Unfinished
- Parser/AST Generator
- Code Minifier
- Code Beautifier
- Code Optimizer
## Version History
- (CURRENT) Adam: Included a full Luau lexer
## Tool Version History
- Lexer/Token Generator: 0.1.0
## Building
- Step 1: `cmake -S . -B build`
- Step 2: `cmake --build build`
- Step 3: Extract the executable from the build folder, should be called as the project name.
- Step 4: Run the executable, read the instructions for usage
## Facts
- The naming for each version released is linear with biblical names, once this project reaches maximum stability, the final version will release and it will be named "Jesus" and will never be updated again.
- This project was made while I was still learning low-level languages like C for the first time or relearning low-level concepts that I forgot long ago, and thats probably why the Lexer was horrible, because this is my first time writing large-scale C projects.
- This project was initially called "Adam" but I quickly realized that the name had zero correlation with what project I was building this for. I eventually settled for Lunaue which is the combination of Luna, and Banaue, which is a rice terraces farm in the Philippines, where I come from.
